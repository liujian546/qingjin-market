# 网络访问配置说明

## 局域网访问
同一局域网内的设备可以直接通过以下地址访问校园交易平台：
- http://192.168.43.6:5000

## 外部网络访问配置

要让不在同一网络的用户也能访问该网站，需要进行以下配置：

### 1. 路由器端口转发设置

1. 登录到您的路由器管理界面（通常在浏览器中输入 192.168.1.1 或 192.168.0.1）
2. 找到"端口转发"、"虚拟服务器"或类似名称的设置选项
3. 添加新的端口转发规则：
   - 外部端口：5000
   - 内部IP地址：192.168.43.6（您的服务器IP）
   - 内部端口：5000
   - 协议：TCP（或选择"全部"）
4. 保存设置

### 2. 防火墙配置

确保Windows防火墙允许5000端口的入站连接：
1. 打开"Windows Defender 防火墙"
2. 点击"高级设置"
3. 选择"入站规则"
4. 点击"新建规则"
5. 选择"端口"，点击"下一步"
6. 选择"TCP"，输入"5000"，点击"下一步"
7. 选择"允许连接"，点击"下一步"
8. 选择适用的网络类型（域、专用、公用），点击"下一步"
9. 输入规则名称"校园交易平台"，点击"完成"

### 3. 获取公网IP地址

1. 在电脑上访问 https://whatismyipaddress.com/ 或其他IP查询网站
2. 记录显示的公网IP地址

### 4. 访问网站

外部用户可以通过以下地址访问：
- http://[您的公网IP]:5000

例如：如果您的公网IP是 123.45.67.89，则访问地址为：
- http://123.45.67.89:5000

## 注意事项

1. 如果您的网络使用动态IP，公网IP可能会变化
2. 某些ISP可能会阻止家用网络的服务器端口
3. 建议使用动态DNS服务来解决IP变化问题
4. 此配置仅适用于开发测试环境，生产环境请使用专业服务器部署
